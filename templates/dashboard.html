About

  • The Hand Tracking Project is a unique initiative to leverage the power of computer vision and machine learning technologies to track human hand movements in real time. The primary aim of this project is to create interactive experiences by enabling machines to understand and respond to human gestures.
  • Developed by Bui Dang Loc, this project uses MediaPipe - a cross-platform, customizable ML solutions for live and streaming media, along with Python and its associated libraries. One of the key features of the project includes the 'Virtual Drawing Board', a fun application where the user can draw on a blank canvas using gestures. The application also allows the user to change colors and clear the canvas using different hand gestures.
  • The project signifies a step towards more natural, intuitive interfaces that allow humans to interact with machines in a way that feels instinctive and seamless

Usage

  • After installing, you can run the project with the following command:
    python main.py
  • Hand Tracking: This module allows the tracking of hand movements in real time.
  • Virtual Drawing Game: This is an interactive game where users can draw on a blank canvas using their hand gestures.
